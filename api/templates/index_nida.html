Garmin Device Fitness Data

This platform provides a comprehensive overview of fitness activity data collected from Garmin devices. It offers a range of API endpoints that allow users to query different aspects of the data, such as activity types, individual activities, performance metrics, and more.

Types of Activities

This endpoint provides data on the types of activities undertaken. Access endpoint at /api/activity-types.

Activities

These endpoints provide data on the activities undertaken with information about the type of activity, duration, and more. Dates should be provided in the format YYYY-MM-DD. Access endpoints at /api/activities/all and others as documented.

Performance Metrics

These endpoints provide performance data on the activities undertaken such as length of time, calories burned, and more. Access endpoints at /api/performance-metrics/all and others as documented.

Lap Metrics

These endpoints provide lap data for running activities. Access endpoints at /api/lap-metrics/all and others as documented.

Elevation Metrics

These endpoints provide elevation data for running activities. Access endpoints at /api/elevation-metrics/all and others as documented.
